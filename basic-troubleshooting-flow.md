## Basic Troubleshooting Flow

**1. Identify the problem**
- Ask what is not working
- Confirm error messages or symptoms
- Determine when the issue started

**2. Check the obvious**
- Power, cables, connections
- Network status
- User permissions or login status

**3. Isolate the cause**
- Try to reproduce the issue
- Check system status and logs
- Compare with a working system if possible

**4. Apply a fix**
- Restart services or system
- Adjust settings
- Apply known solutions

**5. Verify and document**
- Confirm the issue is resolved
- Note what was done for future reference
